# Oasis Infobyte Android Internship Projects

This repository contains Android applications developed during my Android App Development Internship at Oasis Infobyte.

## Projects

### 1. To-Do Application
A task management application where users can add, view, and delete tasks with login functionality.

### 2. Calculator Application
A simple calculator that performs basic arithmetic operations.

### 3. Quiz Application
A multiple-choice quiz app that displays the final score.

### 4. Stopwatch Application
A stopwatch app that measures elapsed time with start, stop, and hold functions.

## Technologies Used

- Kotlin
- XML
- Android Studio
- SQLite
